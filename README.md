# API

This repository contains the backend API for the CS-Space project, a personalized, AI-powered roadmap application for computer science skills.

**Access the API:**
    -   **API Root:** [http://localhost:8000/](http://localhost:8000/)
    -   **Interactive Docs (Swagger):** [http://localhost:8000/docs](http://localhost:8000/docs)
    -   **Neo4j Database Browser:** [http://localhost:7474/](http://localhost:7474/)
